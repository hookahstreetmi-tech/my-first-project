# Snapshot #1: Концепция "Джарвис" (27.01.2026)

## Описание проекта
Создание единой экосистемы автономных агентов на базе N8N, которая автоматизирует путь от идеи до реализации продукта. Платформа: **Indus.8M**.

## Ключевые цели
*   Интеграция разрозненных воркфлоу (Ресерч, SMM, Бизнес-ассистент).
*   Создание Центрального Хаба данных в Supabase для хранения контекста проектов.
*   Автоматизация "логического хода" событий: запрос -> исследование -> анализ -> прототип.

## Текущий статус
1.  **Проанализированы воркфлоу**: `Research Team` (Perplexity), `Pain Points Hunter` (Supabase + Telegram), `Армия SMM`, `Бизнес ассистент`.
2.  **Настроена среда**:
    *   `scripts/fast-push.sh` — автоматизация Git.
    *   `scripts/approve.sh` — интерфейс для подтверждения команд через PowerShell alert в Windows.
3.  **Спроектирован Хаб**: Определены таблицы для Supabase (проекты, знания, гипотезы, контент).

## Что обсуждалось
- Необходимость "умного" ресерчера, который сам уточняет поисковые запросы.
- Создание "охотника за болями", который кластеризует проблемы бизнеса.
- Проблемы с кнопкой "Approve" в интерфейсе и их решение через локальный алерт.
